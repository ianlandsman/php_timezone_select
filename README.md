PHP Timezone Select
===================

An HTML select list and JSON file of PHP timezones. I've cleaned up a few things and added a few niceties as well.

Based on the timezone selection we did for [Snappy](http://www.besnappy.com).

MIT License

You're welcome!

## JSON Format
```json
{
   "AF":{
      "name":"Afghanistan",
      "timezones":{
         "Asia\/Kabul":{
            "abbr":"AFT",
            "offset":16200,
            "name":"Kabul"
         }
      }
   },
}
```
